# Hasta La Vista, Money!

Приложение для учёта личных финансов: счета, бюджет, расходы/доходы, кредиты и кассовые чеки с распознаванием по QR-коду.

## Language

### Загрузка чеков

**Сканировать QR**:
Вкладка страницы загрузки чека, в которой QR-код распознаётся в реальном времени через камеру устройства.
_Avoid_: сканирование камерой, живое сканирование, скан-режим

**Файл**:
Вкладка страницы загрузки чека, в которой пользователь загружает готовое изображение (фото) с QR-кодом чека.
_Avoid_: загрузка изображения, загрузка фото
